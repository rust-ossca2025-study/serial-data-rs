# Week2

## 플러그인

### 공통 특성

- 플러그를 만들 때 가장 먼저 할 일은 사용할 각 입력 데이터의 공통적인 특성을 뽑아 내는 것
- 발견된 특성 나열
  - 사용자 ID
    - 길이 4글자
    - 숫자나 알파벳으로 이루어짐
  - 제품 ID
    - 길이 8글자
    - 숫자나 알파벳으로 이루어짐
- 공통 특성 추출
  - 데이터의 특성
    - alphanumeric 으로 이루어짐
    - 길이 값이 다르더라도, 길이가 미리 정해져 있다는 것 자체가 중요한 공통 특성
  - 데이터 를 처리하는 방법
    - 입력받는 방법이 같음
